# My Own Portfolio

A modern, immersive portfolio website built with React and Tailwind CSS. It features a cinematic video background, glassmorphism UI, and a "click-to-enter" interaction to handle browser autoplay policies.

## Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Icons:** React Icons
- **Deployment:** Vercel

## Features

- **Cinematic Intro:** "Click to Enter" overlay to ensure audio playback.
- **Video Background:** Full-screen looped video with zoom/crop handling.
- **Glassmorphism:** Frosted glass UI effects using Tailwind backdrop filters.
- **Responsive:** Perfectly centered layout on all screen sizes.

## How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/AthanasiosDavaris/portfolio.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```
